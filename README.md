🎯 Objective

To normalize text data by converting different word forms into a single meaningful base form, making Natural Language Processing tasks more accurate and efficient.

🔍 Example
Original Word	Lemmatized Word
mice	mouse
geese	goose
running	run
better	good
children	child
🚀 Key Features
Word Normalization
Context-Aware Text Processing
Lemmatization using WordNet
Vocabulary Reduction
NLP Text Cleaning
Improved Text Consistency
